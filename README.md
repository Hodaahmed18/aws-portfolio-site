# AWS S3 Portfolio Site

A personal portfolio website hosted on AWS S3 with CloudFront CDN for HTTPS and global content delivery.

## What I built
- Static website hosted on AWS S3
- CloudFront distribution for HTTPS and fast global loading
- Bucket policies configured for public access
- Entire infrastructure deployed via AWS CLI on RHEL EC2

## Tech Stack
- AWS S3
- AWS CloudFront
- AWS CLI
- RHEL (Red Hat Enterprise Linux)
- Bash
